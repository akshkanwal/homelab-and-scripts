# Azure Labs

Hands-on Azure lab work supporting AZ-104 (Microsoft Azure Administrator) certification prep and general cloud infrastructure skill development.

Labs are built in a personal Azure subscription and documented with objectives, steps, outcomes, and exam relevance.

## Planned Lab Areas

- Virtual Machine deployment, sizing, and management
- Virtual Networks, subnets, peering, and routing
- Network Security Groups — inbound/outbound rule design
- Azure Active Directory / Entra ID — users, groups, RBAC
- Azure Storage — blob, file, access tiers, SAS tokens
- Azure Monitor — metrics, alerts, Log Analytics workspaces
- Azure Backup — Recovery Services Vault, backup policies
- WatchGuard Firebox on Azure VM — PCNSE lab environment

Lab write-ups will be committed here as they are completed.
