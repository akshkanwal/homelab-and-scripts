# PowerShell Scripts

PowerShell scripts written for IT automation, Active Directory administration, Microsoft 365 management, and deployment tasks.

Scripts are written with clarity and reusability in mind — each script includes a header comment block explaining its purpose, parameters, prerequisites, and example usage.

## Planned Content

- Active Directory: bulk user creation, account management, group membership reporting
- Microsoft 365: license assignment, mailbox management, conditional access reporting
- Endpoint management: Intune enrollment status, compliance reporting
- Utility scripts: log parsing, file management, reporting

Scripts will be committed here as they are written and tested.
