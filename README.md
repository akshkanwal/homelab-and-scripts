# Homelab & Scripts

A living repository of hands-on lab work, Azure experiments, and scripting practice — updated continuously as Aksh progresses through the AZ-104 certification track, WatchGuard PCNSE preparation, and ongoing cybersecurity homelabs.

Scripts and lab notes are committed directly via Git as they are completed. Each subfolder grows over time with real lab output — nothing here is placeholder or theoretical.

## Repository Structure

| Folder | Focus |
|---|---|
| [azure/](./azure/) | Azure VM labs, VNet configurations, AZ-104 prep work |
| [powershell/](./powershell/) | PowerShell scripts for automation and administration |
| [active-directory/](./active-directory/) | AD lab configurations, GPO experiments, identity work |
| [networking-labs/](./networking-labs/) | WatchGuard, SonicWall, VLAN, and network configuration labs |

## Current Focus

- **AZ-104 Azure Administrator** — building labs covering VMs, VNets, NSGs, RBAC, storage, monitoring, and backup
- **WatchGuard PCNSE** — deploying WatchGuard on Azure VMs to practice firewall policy, NAT, VPN, and security services
- **PowerShell** — scripting for Active Directory, M365 administration, and deployment automation

## How This Repository Grows

Every lab completed, every script written, every configuration tested gets committed here with a write-up explaining the objective, the steps taken, and what was learned. This is a documentation-first approach to homelabbing — the work is only complete when it is written up well enough that someone else could follow it.
