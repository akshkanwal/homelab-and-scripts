# Networking Labs

Network configuration labs covering WatchGuard, SonicWall, VLAN design, and general network engineering scenarios.

These labs support both the **WatchGuard PCNSE** certification and general infrastructure skill development, running alongside the Azure labs in a connected homelab environment.

## Planned Content

- WatchGuard Firebox on Azure VM — initial setup, policy configuration, NAT rules
- WatchGuard VPN — Branch Office VPN (BOVPN), Mobile VPN with SSL
- SonicWall configuration scenarios — policy design, security services, VPN
- VLAN design and inter-VLAN routing — Layer 3 switch and router-on-a-stick approaches
- Network segmentation — separating guest, production, and management traffic
- Firewall rule analysis and optimization

Lab notes will be committed here as configurations are built and tested.
